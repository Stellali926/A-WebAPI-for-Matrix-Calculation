Welcome to the Matrix Calculation RESTful API

The Introduction of the Functions:

Store A Matrix

Using POST/PUT call with route "/matrices/(the name of the matrix you stored)"to store one matrix. The example json format of the matrix is:

{"datatype": "Matrix","name": "A","data":{"numrows":2,"numcols":2,"matrixdata":[[2,2],[2,2]]}}

Display A Matrix

Using GET call with route "/matrices/(the name of the matrix you want to display)".

Delete A Matrix

Using DELETE call with route "/matrices/(the name of the matrix you want to delete)".

Matrix Calculation

Using POST call with route "/matrices/calculations" to do matrix calculations.

We support three types of calculations:

  • add
  • subtraction
  • multiplication

You can indicate the operation in the json input. The example json format of the matrix is:

{"datatype": "operation","operationtype": "add","operand1": "A","operand2": "B","resultant": "C"}
